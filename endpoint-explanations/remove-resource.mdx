---
title: 'Remove a Resource'
description: 'Learn how to remove a resource from a collection using the Wetrocloud SDK'
---

Use the `delete_resource` method from the `collection` module to remove resources that are no longer needed from your collections.

### Required Parameters
| Parameter      | Type   | Description                           |
|-----------------|--------|---------------------------------------|
| `collection_id` | String | The unique identifier for the collection |
| `resource_id`   | String | The unique identifier of the resource to delete |

### Request Example
<CodeGroup>
```python Python
from wetro import Wetrocloud

client = Wetrocloud(api_key="your_api_key")
rag_client = client.rag

# Create a collection
rag_client.create_collection(collection_id="research_paper")

# Insert a web resource
insert_response = rag_client.collection.insert_resource(
    collection_id="research_paper",
    resource="https://medium.com/@wetrocloud/why-legal-tech-needs-wetrocloud-ai-rag-and-the-future-of-legal-practice-66fb38c4df09", 
    type="web"
)

# Delete the resource from the collection
delete_response = rag_client.collection.delete_resource(
    collection_id="research_paper",
    resource_id=insert_response.resource_id
)
print(delete_response)
```
</CodeGroup>

### Response

The method returns a response object containing:

```json
{
  "success": true
}
```

| Field      | Description                                    |
|------------|------------------------------------------------|
| `success`  | Indicates if the deletion was successful       |
