The `scrape` endpoint allows you to retrieve specific data from a web page in a structured JSON format based on your predefined schema.
This endpoint supports just one response formats to suit different use cases:  

## Structured Output Response

Structured output is a feature that allows large language models (LLMs) to generate responses in a specific format with your resources. Structured output is best for extracting structured data to fit your applicationâ€™s needs. It returns data formatted with the `json_schema` and `json_schema_rules` you provide.
To use the structured output feature you need to provide the following parameters:

| Parameter           | Type   | Description                                                                 |
|----------------------|--------|-----------------------------------------------------------------------------|
| `website`            | String | The website url to extract from.                                            |
| `json_schema`        | String | The desired structure for the JSON response.                                |

### `json_schema`

The `json_schema` parameter defines the structure of the response you want the system to return. This parameter helps ensure the output matches your application's requirements.

**Example JSON Schema**

```json
{
  "name":"<name of rich man>", 
  "networth":"<amount worth>"
}
```

**Example JSON Schema (List)**

If you want a list of items and not just a single item, you can specify the json schema by putting the entire object in a list

```json
[
  {
    "name":"<name of rich man>", 
    "networth":"<amount worth>"
  }
]
```

**Example Requests**

<CodeGroup>
```bash cURL
curl --request POST \
 --url 'https://api.wetrocloud.com/v1/scrape/' \
 --form 'website="https://www.forbes.com/real-time-billionaires/#7583ee253d78"' \
 --form 'json_schema=[{"name":"<name of rich man>", "networth":"<amount worth>"}]' \
 --header 'Authorization: Token <api-key>'
```
```python Python
import requests
import json

form_data = {
    'website': "https://www.forbes.com/real-time-billionaires/#7583ee253d78",
    'json_schema': json.dumps([{"name":"<name of rich man>", "networth":"<amount worth>"}])
}

response = requests.post(
    'https://api.wetrocloud.com/v1/scrape/',
    data=form_data,
    headers={
        'Authorization': 'Token <api-key>'
    }
)

try:
    print(response.json())
except ValueError:
    print(response.text)
```


```javascript Javascript
const axios = require('axios');
const FormData = require('form-data');

const formData = new FormData();
formData.append('website', 'https://www.forbes.com/real-time-billionaires/#7583ee253d78');
formData.append('json_schema', JSON.stringify([{"name":"<name of rich man>", "networth":"<amount worth>"}]));

axios.post('https://api.wetrocloud.com/v1/scrape/', formData, {
 headers: {
   'Authorization': 'Token <api-key>',
   ...formData.getHeaders()
 }
})
.then(response => {
 console.log(response.data);
})
.catch(error => {
 console.error('Error:', error);
});
```
