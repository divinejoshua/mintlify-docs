Collections in Wetrocloud store indexed resources for retrieval and query operations.

## When to Use the List Collection Method
To get the list of all your collections, use the `get_all` method from the `collection` module in the Wetrocloud SDK. There is no required parameter.

<Note>
This list has a pagination of up to <b>20</b> items per request. `Next` and `Previous` urls will be provided on every response
</Note>
---

### Request Example
<CodeGroup>
    ```python Python
    # Access Modules
    client = Wetrocloud(api_key="your_api_key")
    rag_client = client.rag

    # Get all collections
    all_collections = rag_client.get_collection_list()

    print(all_collections)
    ```

    ```javascript JavaScript
    const axios = require('axios');

    axios.get('https://api.wetrocloud.com/v1/collection/all/', {
      headers: {
        'Authorization': 'Token <api-token>'
      }
    })
    .then(response => {
      console.log(response.data);
    })
    .catch(error => {
      console.error('Error:', error);
    });
    ```
</CodeGroup>

---
