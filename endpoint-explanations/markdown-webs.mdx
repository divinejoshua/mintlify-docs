## How to Generate Markdown for Websites

Make a `POST` request to the `/v2/markdown-converter/` endpoint with the required parameters in the body and make sure you use the `web` type.

<Note>
The resource should be a valid URL and make sure you use the `web` type.
</Note>


### Request Example
<CodeGroup>
 
```python Python
from wetro import Wetrocloud
    
# Initialize the Wetrocloud client
client = Wetrocloud(api_key="your_api_key")
    
# Generate Markdown
markdown_response = client.markdown_converter(
  link = "https://medium.com/@AlexanderObregon/a-brief-history-of-artificial-intelligence-1656693721f9#:~:text=In%20this%20article%2C%20we%20explore,learning%20are%20breaking%20new%20ground.", 
  resource_type = "web"
)

print(markdown_response)
```

```javascript JavaScript
import Wetrocloud from "wetro-sdk";

// Initialize the Wetrocloud client
const client = new Wetrocloud({
  apiKey: "your_api_key"
});

// Generate Markdown
const response = await client.markDownConverter({
  resource: "https://medium.com/@AlexanderObregon/a-brief-history-of-artificial-intelligence-1656693721f9#:~:text=In%20this%20article%2C%20we%20explore,learning%20are%20breaking%20new%20ground.",
  resource_type: "web"
});

console.log("converted to markdown", response);
```

```bash cURL
curl --request POST \
  --url https://api.wetrocloud.com/v2/markdown-converter/ \
  --header 'Authorization: Token <api-key>' \
  --header 'Content-Type: application/json' \
  --data '{
    "link": "https://medium.com/@AlexanderObregon/a-brief-history-of-artificial-intelligence-1656693721f9#:~:text=In%20this%20article%2C%20we%20explore,learning%20are%20breaking%20new%20ground.", 
    "resource_type": "web"
  }'
```
</CodeGroup>
