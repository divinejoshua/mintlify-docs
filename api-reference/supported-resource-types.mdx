---
title: 'Supported Resource Types'
description: 'Learn about the resources wetrocloud API supports'
---
By default, Wetrocloud API supports 4 major types of resources:

| **Resource** | **Description**                                                                                                            |
|--------------------|:----------------------------------------------------------------------------------------------------------------------------|
| **File**           | Supports various file types including `.csv`, `.docx`, `.epub`, `.hwp`, `.ipynb`, `.jpeg`, `.jpg`, `.mbox`, `.md`, `.mp3`, `.mp4`, `.pdf`, `.png`, `.ppt`, `.pptm`, `.pptx`. |
| **Text**           | Plain text content.                                                                                                       |
| **JSON**           | Structured data in JSON format.                                                                                           |
| **Web**       | Web-based resources, such as websites.                                                                |

<Note> To use the `file` type, all files must be provided as URLs. You can consider services like Amazon S3, Cloudinary, or similar platforms to host your files and provide their URLs to the API. </Note>
